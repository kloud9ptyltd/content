Vibe Trading

Vibe-code an AI trading desk — on your own laptop.

Describe a strategy in plain English and a team of AI agents spins up — a bull, a bear, a risk officer, a portfolio manager — that pull live market data, run backtests, and actually debate the trade before a final call. Free, open-source, runs locally.

Setup — 10 minutes

1 Prerequisites

You need Python 3.11+. Check with python3 --version. On a Mac, brew install python@3.11; on Windows grab it from python.org.

→ github.com/HKUDS/Vibe-Trading — the open-source project (star it to say thanks).

2 Install it

One command from PyPI:

pip install vibe-trading-ai
This gives you three commands: vibe-trading (CLI), vibe-trading serve (web app), vibe-trading-mcp (plug into Claude Desktop / Cursor).

3 Get an LLM key — this is the part everyone gets wrong

The agents need a language model to think. We tested every free option so you don't have to:

Free tiers can't run it. The agents send big prompts and fire lots of calls — Groq's free tier is too small, Gemini free throttles to 5 requests/min and its newest models break tool-calling. It also doesn't support Claude directly. The cheap sweet spot is DeepSeek: a full 4-agent run costs ~1–3¢, so a couple of dollars lasts for dozens of runs.

Create a key at platform.deepseek.com → Top Up (~$2 is plenty) → API keys → copy it (starts with sk-).

4 Point it at DeepSeek

Run the interactive setup and choose DeepSeek when asked, then paste your key:

vibe-trading init
Behind the scenes it just sets: provider deepseek, model deepseek-v4-flash, base URL https://api.deepseek.com/v1. Market data (yfinance, OKX, and more) works with no extra keys.

5 Run it in your browser

Start the web app — leave this terminal window open, it's the server:

vibe-trading serve --port 8899

Then, in any browser (Chrome, Safari, Edge):

1. Go to http://localhost:8899
2. Click Dark (bottom-left) for the terminal look
3. Open the Agent tab in the left sidebar
4. Type a prompt in the box and press Enter — watch the agents pull live data, think, and reply

Keep the terminal running the whole time — closing it stops the server, and the browser tab goes blank. To stop later, press Ctrl+C in that terminal. Prefer the terminal? Skip the browser and run the commands below directly.
Try these first

🐂🐻 Run a full investment committee

Four agents — bull, bear, risk officer, PM — research the name, argue it out, and the PM makes the final call. This is the flagship. Fastest in the terminal:

vibe-trading --swarm-run investment_committee '{"target":"NVDA","market":"US equities"}'
Swap "NVDA" for any ticker. See all 29 teams with vibe-trading --swarm-presets.

💬 Prompts to paste in the Agent tab

Analyze NVDA: pull recent price data, assess momentum, run a 20/50-day moving-average backtest, and summarise bull vs bear.
Research MP Materials — momentum plus the rare-earth / US supply-chain thesis.
Backtest a dual moving-average crossover on Bitcoin for the last year and report return and drawdown.

! Good to know

Keep it paper / research. It can connect to brokers, but leave that off — use it to research and backtest, not to place live trades.

Your keys stay on your machine. The app runs locally; only the model calls go out to your chosen provider.

AI can be confidently wrong. The agents produce a plausible case every time — that's the point of the debate, not a guarantee. Treat every verdict as a starting point, not a signal. Size your own risk.
