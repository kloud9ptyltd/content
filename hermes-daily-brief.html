What it actually does

☀️ A brief in your pocket, every morning

Hermes has a built-in scheduler. Tell it — in plain English — to send you a market brief at 7am, and every morning it pulls the tape, the headlines and your holdings, then writes a tight, human read and delivers it to your chat app. No dashboard to open.

“Morning. Tech's soft but your book's holding — Nvidia up 4%, doing the heavy lifting. Iran's rattling the Strait of Hormuz, so watch crude. Portfolio: +0.7%.”

💼 Ask it about your own stocks

Because it remembers your portfolio across sessions, you can just message it — “how's my book looking?”, “is it worth trimming Nvidia?”, “what's the risk in my tech names this week?” — and get a grounded answer using live data, wherever you are.

Setup — about 10 minutes

1 Install Hermes

One line. The installer sets up the agent, its managed environment and the hermes command. Works on macOS and Linux (a $5 VPS is plenty; it barely uses anything when idle).

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
Built by Nous Research · source: github.com/NousResearch/hermes-agent

2 Run the setup wizard

This walks you through everything at once — pick a model provider and drop in a key. OpenRouter is the cheap, easy pick (one key, hundreds of models); you can also use Nous Portal, OpenAI or your own endpoint.

hermes setup
Swap models anytime with hermes model — no code changes, no lock-in. A cheap model is perfectly good for a daily brief.

3 Talk to it once from the terminal

Make sure the brain works before wiring up chat. Launch the terminal UI and say hello — then tell it who you are and hand it your watchlist.

hermes
Try: “Remember my portfolio: Apple, Google, Nvidia, Microsoft, Amazon.” It keeps that across sessions.

4 Connect a chat app

One gateway process serves Telegram, Discord, Slack, WhatsApp and Signal. Run the setup, start the gateway, then just message your bot.

hermes gateway setup hermes gateway start
After this, sending your bot a message is the same as chatting in the terminal — same agent, same memory, from anywhere.
Wiring up each channel

✈️ Telegram · easiest

In Telegram, message @BotFather, run /newbot, and copy the token it gives you. Paste it when hermes gateway setup asks. Done — DM your new bot and it answers.

🎮 Discord

Create an app at the Discord Developer Portal, add a Bot, copy its token, and invite it to a private server or DM. Feed the token to the gateway setup and lock it to your user so only you can talk to it.

💬 WhatsApp

The gateway prints a QR code — scan it from WhatsApp (Linked Devices), the same way WhatsApp Web works. After that, texts to that number reach your agent.

Full per-platform docs: hermes-agent.nousresearch.com/docs → Messaging Gateway.
