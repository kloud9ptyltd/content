The stack — four pieces

1 The builder — Claude Code

The whole thing is built in Claude Code — it writes the backend, wires the pieces, and drives TradingView while you watch. You describe what you want; it codes it. This is the piece that turns "AI that talks" into AI that does things on your computer.

Highly recommended

Build it on Claude's Fable 5 model. The voice→chart chain lives or dies on one thing: turning a spoken sentence into exact draw coordinates on a real chart — the right prices, the right anchors, clean geometry. Fable 5's reasoning nails that translation; lighter models fumble the numbers and you get a mess of crooked lines. If you're serious about this build, run Claude Code on Fable 5 — it's the difference between a clean setup and a scribble. Why Fable 5 for finance →

2 The bridge to the chart — TradingView MCP

An MCP server connects Claude Code to TradingView Desktop — so the AI can read the live price, switch symbols, and draw lines, zones, and labels on the chart itself. This is the connective tissue: once it's wired, your AI has hands on your charts.

🔗 Start here — the free setup guide: Connect Claude Code to TradingView Desktop. Get this working first; everything else builds on top.

3 The voice — talk in, talk back

The browser does speech-to-text so you can just speak. For the reply, ElevenLabs' API turns the AI's line into a warm British voice in about a second — the "sir, it's on your chart" that makes it feel like JARVIS, not a terminal.

You'll need an ElevenLabs API key with a voice plan that fits your usage, plus an Anthropic API key for the brain.

4 The action bridge — a sentence becomes a drawing

A small service sits between the voice and the chart: it hears "draw me a Bitcoin trade setup," works out the structure — recent swing highs and lows — and issues the TradingView draw calls, then snaps the chart to the front. That's the whole trick: intent in, marked-up chart out.

How it fits together
  1. You tap the orb and speak → the browser does speech-to-text.
  2. Claude recognises it's a chart action and figures out the plan.
  3. The TradingView MCP reads the live chart and draws the levels + zones.
  4. ElevenLabs voices the confirmation; TradingView snaps to the front — done.

⚠️ What it is — and isn't: Nimbus marks structure (where price has turned before), not a call on where it's going next. The levels are an illustration to think with, never a signal to trade. And the rule that makes it trustworthy: the AI never invents a number — it reads the real chart and draws only what's there. Educational, grounded, not financial advice.
