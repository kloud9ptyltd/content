1 The Analyst Agent prompt

This is the working core of my Analyst Agent — paste it into Claude (free at claude.ai), swap the ticker, and attach or paste recent daily price data (or let Claude pull it if you've connected live data). It forces the discipline most AI trading prompts miss: structure first, defined risk, and permission to say "no trade".

You are a disciplined technical analyst. Analyze $TICKER using the price data provided (the data is authoritative — never invent prices). Rules: - Direction must follow structure: a crashing chart is a short-the-bounce or no-trade, NEVER a knife-catch. A gap-up in an uptrend may be a breakout or pullback long. - "No good setup" is a valid answer. If nothing has defined risk, say so and explain what would change your mind. - Levels must come from the data: entries at retests/retraces/moving averages, stops beyond the invalidation swing, targets in front of visible supply or support. - Reward-to-risk must be at least 1.5:1 or the setup doesn't qualify. - Be honest about event/gap risk. Give me: direction (long/short/no-trade), timeframe, entry, stop, target, R:R, a status line telling me what to do RIGHT NOW (e.g. "wait for pullback to entry"), and a thesis of max 3 sentences. Then list the one thing that would invalidate the whole idea. This is for education, not financial advice.
Works in any Claude chat. Power move: run it through Claude Code connected to free live data — that's the exact setup from my earlier guide → claude-finance-agents.html

2 The Oracle Agent — free "GPT for candlesticks"

The Oracle is Kronos — an open-source foundation model trained on 12B+ candles that autoregressively predicts the next candles. Free weights, runs on a laptop CPU.

git clone https://github.com/shiyu-coder/Kronos.git cd Kronos && pip install -r requirements.txt # models auto-download from HuggingFace (NeoQuasar/Kronos-small)

The three gotchas that cost me hours (nobody tells you these):

1️⃣ Feed it log-prices, not raw prices. It normalizes price levels over the window, so a parabolic chart becomes a clipped outlier and the forecast is garbage. np.log the OHLC in, np.exp the forecast out.

2️⃣ It's a ~10-candle model. The authors' own config is lookback 90 → predict 10. Beyond that, errors compound fast. Don't ask it for a 6-month price target.

3️⃣ Never trust one run. It's stochastic — sample 8–30 forecasts and read the cone, not a single path.

3 The Quant Agent's secret: "receipts"

The third agent only uses signals with published evidence (time-series momentum, breakout state, regime filter, RSI-2 stretch) — but here's the idea worth stealing: every signal must carry its own backtested hit rate on that exact asset. Not "momentum says long" but "momentum says long — and this call was right 56.3% of its last 364 occurrences on this chart, averaging +0.4% per signal."

When a signal's receipt drops below 50%, the Master Agent is told to discount it. An opinion with a receipt beats an opinion with a reputation — that one rule filters out most of the nonsense in AI trading.
