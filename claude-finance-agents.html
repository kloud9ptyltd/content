Turn Claude into a Wall Street analyst — free.

Anthropic just shipped 10 finance AI agents (the work banks pay six figures for). Here's the exact setup we used to install all 10, wire free live market data, and have it analyze any stock in seconds — no paid data, no API key.

📊 Equity Research🧮 Model Builder📈 Market Researcher 🎯 Earnings Reviewer💼 Pitch Builder🔎 Valuation Reviewer + 4 more

1 Get Claude Code

Install the Claude Code CLI (free with any Claude plan). It runs the agents on your machine.

→ claude.com/code

2 Add Anthropic's finance marketplace

claude plugin marketplace add anthropics/financial-services

3 Install the agents

Equity research is the one for stocks (gives you /thesis, /screen, /earnings). Add the others you want.

claude plugin install equity-research@claude-for-financial-services claude plugin install financial-analysis@claude-for-financial-services claude plugin install market-researcher@claude-for-financial-services

4 Wire FREE live data (no API key)

This connects Claude to real market prices via Yahoo Finance — zero signup.

claude mcp add yahoo-finance -- uvx mcp-yahoo-finance
Want real-time streaming too? A free Finnhub key (finnhub.io) works — but the above is enough to start.

5 Analyze any stock

In Claude Code, just run the thesis command on a ticker — it pulls live data and writes the full report.

/thesis MU
…or paste this full prompt (swap the ticker):
Use the equity-research agent with live market data to write a full investment thesis on $MU. Include: current price + valuation (P/E and forward P/E), revenue & margin trends, the bull case (3-5 pillars), the bear case (key risks), segment growth, analyst consensus, and a one-line verdict. Cite real numbers from live data and cross-check them. This is not financial advice.
